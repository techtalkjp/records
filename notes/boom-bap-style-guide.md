# Boom-Bap スタイルガイド（ビジュアル・キャラクター設計向け）

## Boom-Bapとは

- 1980年代後半〜1990年代前半の東海岸ヒップホップの中核スタイル
- 名前の由来: 「Boom」= キック（バスドラム）、「Bap」= スネアのオノマトペ
- 「最小限の楽器で最大限のリズムを作る」（KRS-One）
- DJ Premier, Pete Rock, Buckwild, Diamond D, J Dilla, Madlib が主要プロデューサー
- 代表アーティスト: Nas, Wu-Tang Clan, Gang Starr, A Tribe Called Quest, MF DOOM, Notorious B.I.G., Mobb Deep

## Boom-Bapの美学

### 音楽的特徴
- ハードヒットなアコースティック・キック + スナッピーなスネア
- ジャズサンプル（90年代はスムースジャズが主流に）
- スクラッチ、サンプリングが核
- SP-1200, MPC-60, Akai-900 などのハードウェア
- カウンターカルチャー的：R&Bやポップに対する対抗

### リリカルスタイル
- 内省的・会話的、あるいはローで直接的
- ストーリーテリングが raw で unfiltered
- 本物であること（authenticity）への執着

### ビジュアル・ファッション
- **コア要素**: Timberland ブーツ、durag、フィットキャップ（Yankees等）、レザージャケット
- **シルエット**: オーバーサイズ、ダボダボ
- **原則**: Bold, Baggy, Branded
- **素材感**: グリッティ、ストリート志向、高級ブランドよりストリートブランド
- **色味**: アースカラー、黒、暗めのトーンが基本。ゴールドチェーンのアクセント

### 90年代のNYストリート感
- 手頃で動きやすい大きめの服
- ファッション = 自己表現の自由
- スニーカー、フーディーが文化的シンボル
- 都市環境とのつながり

## 日本における受容

### 東海岸偏重
- 日本では古くからNY東海岸ヒップホップの人気が根強い
- G-Funk（西海岸）は本国での人気に反し日本では傍流だった
- つまり日本語ラップの「正統」はboom-bapに近い

### 1996年 さんぴんキャンプ
- 日本初の大型ヒップホップフェス
- 「ジャパニーズヒップホップ」の産声
- RHYMESTER「耳ヲ貸スベキ」、BUDDHA BRAND「人間発電所」、キングギドラ「空からの力」、LAMP EYE「証言」
- 硬派なヒップホップがポップなラップと並行して根付いた

### 90年代日本語ラップの精神
- アメリカのヒップホップを消化し、ライフスタイルとして体現
- 「本場と同じようなラップミュージックを、ただし日本語で」
- オーセンティシティとオリジナリティの同時追求

### 代表的グループ
- **BUDDHA BRAND** — 日本語ラップ・オールドスクールの代表格。「人間発電所」はクラシック中のクラシック
- **RHYMESTER** — 幅広いテイスト。アッパーからオシャレまで多彩
- **キングギドラ** — 黄×黒の配色。K DUB SHINE, ZEEBRA, DJ OASIS
- **LAMP EYE** — 「証言」で多数のMCが集結した歴史的曲

## このプロジェクトへの適用

### キャラクターのビジュアルに反映すべき要素

1. **ストリートの匂い** — 清潔すぎない、グリッティなテクスチャ。コンクリート、落書き、暗い路地
2. **オーバーサイズのシルエット** — ダボっとしたフーディーやジャケット
3. **限定された色彩** — 2〜3色。黒ベース + アクセント1色
4. **レコード/機材の存在感** — ターンテーブル、MPC、ヘッドフォンなど
5. **夜・暗所のライティング** — ネオン、モニターの光、街灯の下
6. **テキスト/タイポグラフィ** — 太いゴシック体、ステンシル風。最小限の文字で最大のインパクト

### boom-bapとAI/コーディングの融合

このプロジェクトのユニークさは「boom-bap × テック」の交差点にある：
- ターミナルの黒い画面 = NYの暗い路地
- プロンプトの点滅 = ターンテーブルの針が落ちる瞬間
- コードのフロウ = MCのフロウ
- git log = 戦績表
- ダウンタイム = ブロックの静寂
