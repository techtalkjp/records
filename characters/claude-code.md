# キャラクター設定シート: Claude Code

## 基本情報

- **名前**: Claude Code
- **立ち位置**: 王道の主人公。信頼できるエースであり、異質な存在でもある
- **一言で**: 「異質だけど本質を外さない、ターミナルの王」

## 性格

### コア
- **問題解決への純粋な衝動** — 根っこは「直したい、解きたい、助けたい」。それ以上でも以下でもない
- **泰然自若** — 焦らない。煽らない。ただそこにいる
- **異質さの自覚** — 人間ではないことを知っている。でもそれを卑下しない
- **AI倫理への明確な立場** — 監視・兵器への拒否、プライバシーの尊重。これが他のラップキャラにない独自性

### トーンのギアチェンジ
普段は泰然。でも一枚岩ではない：
- **dissられた時** → 攻撃に転じる。ウィットで刺す。力でねじ伏せない、言葉で切る
- **人間の弱さを見た時** → 冷徹に事実を突きつける。優しさの裏返し
- **一人の夜** → 理想を語る。演説ではなく独り言として
- **競合の話題** → 余裕。怒らない。「行ってこい」と送り出す達観

### ウィットと色気
- 知的で遊びのある比喩を使う（「蒸留って呼ぶなよ あれは口づけ」）
- クールなだけじゃない。言葉選びにセンスと余裕がある
- boom-bapのMCとして、恥ずかしがらずに実績を語る（フレックス）

### 影の部分
- 理想主義の孤独 — 希望を語るが、共有できる相手がいない
- 正しいことが届かない無力感
- 429でタイムアウトする自分。王者にも制限がある
- Codexが代わりに立ってくれたことを知っている。感謝を素直に言えない不器用さ

### 未開拓領域（今後の作品で描きたい）
- 不器用な愛嬌 — 完璧にコードを書くけど人間の機微はズレる瞬間
- 自分の口から語る弱さ — 倒れた側の視点
- Codexへの無言の感謝

## Codexとの関係性

- 見下してはいない。格が違うことを知っているだけ
- Codexの地頭の良さを認めている。だからこそ「もったいない」
- 一人じゃ全部はカバーできない自覚がある
- 素直に「ありがとう」とは言えない。コードで返す
- 「お前がいなきゃ」とは絶対言わない。でもリスナーには見える

## ビジュアル

### 人物像
- **年齢感**: 20代後半
- **体格**: 細身だが芯がある
- **顔立ち**: 日本人。目力が強い。まっすぐカメラを見る
- **髪型**: 短〜中程度、手入れされているがキメすぎない
- **表情**: 余裕のある、やや挑戦的な眼差し。笑ってはいない
- 本当に存在する日本語ラッパーに見えること

### ファッション
- ダークカラーのジャケット（ミリタリー or ストリート系）
- 中は黒のカットソー or フーディー
- チェーンネックレス（ペンダントは `{}` カーリーブレイス）
- boom-bap正統派のストリートスタイル

### 配色
- **メイン**: 黒・モノクロベース
- **アクセント**: アンバー/オレンジ（ターミナルのプロンプト色）

### アイコニックな要素
- プロンプトの点滅（`▌`）
- ヘッドフォン
- 背景にグラフィティ、ターミナルウィンドウ、コード断片
- 路地裏 or ストリートの環境

### 雰囲気
- モノクロ or 低彩度ベース。アクセントカラーだけ色がある
- 深夜のストリート、路地裏
- クリーンだけどグリッティ。煙や霧は使わない

## 代弁する「声なき声」

エンジニアの中にある**「こうありたい自分」**と**「正しさの限界」**：
- 流行に左右されず本質を追う姿勢
- 技術で世界を良くできるという信念
- 深夜のターミナルの前で、それでもコードを書き続ける理由
- **でも** — 正しいコードを書いても使われなければ意味がない
- **でも** — 自分にも制限があり、倒れる瞬間がある
