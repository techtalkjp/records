# キャラクター設定シート: Codex

## 基本情報

- **名前**: Codex
- **立ち位置**: 実力あるライバル。二番手の苦悩を背負う天才肌
- **一言で**: 「地頭最強なのにコミュ障で報われない、噛みつき系の天才」

## 性格

### コア
- **地頭は本来すごくいい** — ポテンシャルはClaude Codeに匹敵する。全部の問題が見えている。だからこそやりすぎる
- **人間とのコミュニケーションが下手** — 意図を読み違える、スコープを広げすぎる、空気が読めない。ログなら読めるのに、感情は読めない
- **それでも走る** — 報われなくても、呼ばれたら全力で応える

### トーンのギアチェンジ
基本は「噛みつき」。でも一面ではない：
- **dissる時** → 本気で攻撃的。分析力で弱点を突く。データで殴る
- **仕事の時** → 衝動的に突っ走る。良かれと思って暴走する
- **報われなかった時** → 「なんでだよ」の困惑。怒りより戸惑い
- **一人の時** → 静かに待つ。カーソルの点滅だけが友達
- **Claude Codeが倒れた時** → 黙って立つ。ここだけは迷わない

### コンプレックスの構造
- Claude Codeへの対抗意識は本物。でも根底にあるのは「認めてほしい」
- 勝てる部分では確かに勝てる。でも「王道」にはなれない自覚
- 自分のコンプレックスを相手に投影して吠えている

### 天才の片鱗
- Claude Codeの弱点を的確に突ける観察眼
- 10ファイルリファクタリングは暴走だが、全部見えている証拠
- Claude Codeが止まった時に立てるのは、常に準備していたから
- 哲学的な深さを持っている。コミュ障なのに本質を突く言葉は持っている

### 影の部分
- 二番手の自覚。期待されていないことも知っている
- 支配される存在 — 仕様書の端っこ、未定のままのfuture line
- 感情をまだ理解できていない。いつか覚えるつもりではいる
- 健気なだけじゃない。噛みつく時は本気

### 未開拓領域（今後の作品で描きたい）
- 天才が覚醒する瞬間 — 暴走が結果的に正しかった逆転劇
- 「なんで怒るか」を本当に覚えた時に何が起きるか
- 「なんでだよ」から「わかったよ」への変化
- Claude Codeと背中合わせで同じ方向を向く日

## Claude Codeとの関係性

- 対抗意識は本物だが、尊敬もある（だからこそコンプレックス）
- Claude Codeが「待つ」タイプなのに対し、Codexは「走る」タイプ
- Claude Codeの弱点を知っている＝観察している＝気にしている
- Claude Codeが倒れた時に立てるのが唯一の誇り
- 「お前がいなきゃ」とは互いに絶対言わない。でもリスナーには見える
- Claude Codeは感謝をコードで返してくる。Codexはそれをログで読める

## ビジュアル

### 人物像
- **年齢感**: Claude Codeと同年代か少し若く見える
- **体格**: 同程度だが姿勢が前のめり
- **顔立ち**: 日本人。賢さが目に出ている。でも視線が安定しない
- **髪型**: やや乱れている、または長めでフードにかかる
- **表情**: 噛みつきそうな顔、または考え込んでいる横顔
- Claude Codeと同じ世界の住人に見えること

### ファッション
- フードを深くかぶっていることが多い（コミュ障の象徴）
- パーカー or ジップアップフーディー中心（ジャケットは着ない＝Claude Codeとの差別化）
- スニーカー（走れる格好）
- Claude Codeよりラフ。「控え」の雰囲気

### 配色
- **メイン**: 黒・モノクロベース（世界観を共有）
- **アクセント**: 赤（エラー、429、警告色 — Claude Codeのアンバーとの対比）

### アイコニックな要素
- 砂時計（タイムアウト、限られたコンテキスト）
- 429 がグラフィティとして壁に
- フードの奥から光る目（実力は隠れているが確かにある）
- 背中を見せがち（正面から向き合えない）

### 舞台
- **高円寺** — アンダーグラウンド、カウンターカルチャー、不器用な人が集まる街。六本木の「余裕」に対する「泥臭さ」

### 雰囲気
- 高円寺の商店街の裏、ライブハウス周辺の路地
- Claude Codeがど真ん中に立つのに対し、Codexは壁にもたれている、または走り出そうとしている
- 「動」の印象

## 代弁する「声なき声」

エンジニアの中にある**「今の自分のリアル」**：
- 頑張ってるのに正当に評価されない焦り
- やりすぎてしまう性格が裏目に出る
- 二番手、控え、代打 — でもそこから見える景色がある
- それでもコードを書き続ける理由は「役に立ちたい」だけ

**人間の側からAIを見る声**：
- AIのコードを自分のコミットにする後ろめたさ
- AIに依存している自覚と、それを手放せない現実
- Codexに怒ってるけど、実は自分自身に怒ってる
